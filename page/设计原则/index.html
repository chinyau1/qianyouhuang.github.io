  • 白细胞正常范围
    成人:(4.0-10.0)×10^9个/L、新生儿:(15-20)×10^9个/L、6个月到2岁:(11-12)×10^9个/L、4到14岁:8×10^9个/L、儿童:(5.0-12.0)×10^9/L 。机体免疫应激状态数量会有所升高，感染伤寒杆菌（沙门菌）数量下降。
  • 血红蛋白正常范围
    成年男性:120～160g/L、成年女性:110～150g/L、新生儿:170～200g/L、青少年(儿童):110～160g/L。
  • 癌胚抗原CEA: 正常范围< 5ng/ml
    CEA升高常见于大肠癌、胰腺癌、胃癌、乳腺癌、甲状腺髓样癌、肝癌、肺癌、卵巢癌、泌尿系肿瘤等。吸烟、妊娠期和心血管疾病、糖尿病、肠道憩室炎、直肠息肉、结肠炎、胰腺炎、肝硬化、肝炎、肺部疾病等15%-53%患者血清CEA也会有所升高。因此CEA不是恶性肿瘤的特异性标志，在诊断上只有辅助价值。
  • 肿瘤标志物CA19-9: 正常范围< 37U/ml
    粘蛋白型的糖类蛋白肿瘤标志物，是对胰腺癌敏感性最高的标志物。在血清中它以唾液粘蛋白形式存在，分布于正常胎儿胰腺、胆囊、肝、肠和正常成年人胰腺、胆管上皮等处。对消化道恶性肿瘤如胰腺癌、肝胆系癌、胃癌、结直肠癌等有较好的指示作用，可见于卵巢癌。也有医院说0-40U/ml的。
  • 甲胎蛋白AFP
    成年后为定量检测为阴性，正常放射免疫法 < 25ng/mL，化学发光法 < 5Iu/mL。
  • 总胆红素
    总胆红素正常值一般在21以下，超过21到34之间属于隐性黄疸，超过34属于显性黄疸（仅参考）。目前的总胆红素水平处于显性黄疸的范围，应该有皮肤粘膜黄染表现。完善检查，明确黄疸的原因，常见的有肝前性黄疸，肝细胞性黄疸以及肝后性黄疸。肝前性黄疸即为非结合胆红素生成过多导致的黄疸；肝细胞性黄疸是指因肝细胞受损，对胆红素的摄取、结合以至排泄发生障碍，胆红素在血中蓄积所致的黄疸；肝后性黄疸多由胆总管阻塞所致,故相当于肝外阻塞性黄疸。
  • 谷丙转氨酶ALT
    ALT参考值为男性: 5-40U/L，女性: 5-35U/L。ALT是诊断肝细胞实质损伤的主要项目，其水平通常与疾病的严重程度平行。
